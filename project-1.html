This is a comprehensive project which features both UI/UX and backend along with database integration.

Project Overview

The Face Recognition project builds a smart App which when fed with the url of an image, it detects the faces in it. The project is fully responsive and uses different NPM libraries like Parallex-tilt and Particles for its UX/UI rendering. It uses Clarifai API to detect the faces in the image and labels them. The app also features sign in and register forms which feeds the data to the PostGres database.

I have deployed and hosted the application on Heroku platform which also hosts its database. The project does react state management and efficiently handles all routes. The source code is available on Github and a live version can also be found hosted on Heroku.
