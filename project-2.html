RoboFriends

This short and fun project searchs and filters a list of robots which are fetched using RoboHash's API and are rendered using React components.

Live Link

Project Overview

The RoboFriends project is a single page application which is implemented using HTML, CSS, Javascript, React and also combines API fetch calls. The App also does react state handling and uses NPM to use modules across it.

The project is fully responsive and uses Tachyons library and font files for styling as well. The source code is available on Github and a live version can also be found through Github Pages.
