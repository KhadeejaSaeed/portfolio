BelkaVpn

The project was done in collaboration to integrate front-end and back-end of a Google chrome BelkaVpn extension.

Project Overview

This is a HTML, CSS and JS based project along with API integration. The task was to develop front end for the VPN's google chrome extension, given the PSD file. Along with it, the front-end was made compatible with back-end with API calls. The design was fully responsive and was made keeping extensions and their dimensions in mind.

Node libraries were also used, along with different font families and locale settings according to the regions for widespread compatibility. The project was a collaborative effort and the source code can be found on my github repo.
